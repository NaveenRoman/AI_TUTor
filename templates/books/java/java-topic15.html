import java.util.*;
import java.io.*;

public class StudentConsole {
  static List list = new ArrayList<>();
  public static void main(String[] args) throws Exception {
    list.add(new Student("Naveen",21));
    Scanner sc = new Scanner(System.in);
    while(true){
      System.out.println("1:Add 2:List 3:Exit");
      int c = sc.nextInt(); sc.nextLine();
      if(c==1){ System.out.print("Name: "); String n=sc.nextLine(); System.out.print("Age: "); int a=sc.nextInt(); sc.nextLine(); list.add(new Student(n,a)); }
      else if(c==2){ list.forEach(System.out::println); }
      else break;
    }
  }
}
class Student { String name; int age; Student(String n,int a){name=n;age=a;} public String toString(){return name+" ("+age+")"; } }

Intermediate: Replace in-memory list with file serialization (ObjectOutputStream) or JDBC for persistence.

import java.nio.file.*;
import java.util.*;
import java.util.stream.*;
public class WordCount {
  public static void main(String[] args) throws Exception {
    String text = Files.readString(Paths.get("sample.txt"));
    Map freq = Arrays.stream(text.split("\\W+"))
      .filter(s->!s.isBlank())
      .map(String::toLowerCase)
      .collect(Collectors.groupingBy(w->w, Collectors.counting()));
    freq.entrySet().stream().sorted(Map.Entry.comparingByValue().reversed()).limit(10)
      .forEach(e->System.out.println(e.getKey()+": "+e.getValue()));
  }
}
Simple word frequency analyser
üìò Beginner ‚Üí Pro Explanation

Beginner: Count words using Maps and loops.

Intermediate: Use Streams and handle large files with Files.lines (streaming).

Pro: Extend to Hadoop/Spark for big data, or use tries for prefix queries and memory optimizations.

üí° Pro Tips & Interview Notes
  • Discuss time and space complexity: O(n) time, O(v) space (v = unique words).
  • Show how to handle stopwords, normalization, and memory trade-offs for huge data.
